# 🧠 自然语言处理原理与实战

<h3 align="center"><a href="README.md">中文</a> | <a href="README_en.md">English</a></h3>

> 让机器读懂人话，从零开始的 NLP 之旅

## 这是什么？

一套给**想学 NLP 但不知道从哪开始**的人准备的课程。

不是那种"我给你列个公式你自己推"的课——每节课都有**手写算法实现**，从分词到搜索到深度学习，19 课带你从"什么是 NLP"走到"能自己做一个聊天机器人"。

每节课末尾有练习题，还有参考答案可以对照。学到最后你会明白：为什么搜索引擎能找到你想要的东西，为什么机器能判断一句话是好评还是差评，为什么聊天机器人能听懂你在说什么。

## 怎么跑？

```bash
# 装依赖
pip install jieba gensim scikit-learn numpy torch transformers elasticsearch

# 进入某一章
cd s_nlp/02_中文分词

# 跑演示
python main.py

# 做练习
python exercises.py
```

| # | 你会做什么 | 顺便学了什么 |

## 技术栈

- **语言**: Python 3.8+
- **NLP 工具**: jieba, gensim, sklearn, sklearn-crfsuite
- **深度学习**: PyTorch
- **预训练模型**: transformers (Hugging Face)
- **搜索引擎**: elasticsearch (Python 客户端)
- **数据处理**: numpy, collections

## 几个小建议

1. **别跳章** — 每章的知识是后续章节的基础，跳了会懵
2. **别光看** — 跑完 main.py 后，自己改参数、换数据，看结果怎么变
3. **做练习** — 练习题不是摆设，先自己写再看答案
4. **理解原理** — 不要只记 API 调用，理解算法背后的数学才能灵活用
5. **动手做项目** — 学完后试试用 NLP 解决实际问题，比如情感分析、文本分类

## 课程特色

- 🧮 **手写算法** — 核心算法不用库，从零实现，理解底层原理
- 🎯 **实战导向** — 每课都有可运行的 demo，不是纸上谈兵
- 📝 **练习题** — 每课 3-5 道，有 TODO 桩 + 参考答案 + 测试函数
- 💬 **生活类比** — 复杂概念用日常例子解释，降低理解门槛
